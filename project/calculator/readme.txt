Function: tiny calculator, only +-*/

版本说明：
	calc_v1：该版本实现方法：
				1. 将中缀表达式转换成后缀表达式(逆波兰表达式)
				2. 通过分析后缀表达式，再配合双栈实现运算
	
	calc_v2: 该版本实现方法：
				1. +- */ 连续运算可以通过while循环实现
				2. +-*/  混合运算可以通过函数调用来实现
				3. ()    运算可以通过递归来实现
				具体参考：EBNF规则

使用说明：
	calc_v1:
		1. 输入中缀表达式时，数字-1 和 - 1是不一样的效果
		   如果输入的是负数，如-23, 则后面如果跟减法运算，则需要加空格，如-23 - 4
		2. 每当计算完成后，输入q退出，输入别的键继续，比如ENTER
	
	calc_v2: 
		数字和运算符可以紧挨在一起，也能处理负数
		eg. (2+3*-5)/12-5*4
